# 请求

## 请求地址

```js
    GET https://api.kasuie.cc/mio
```
or 
```js
    POST https://api.kasuie.cc/mio
```

## 请求参数

| 参数名   | 参数类型 | 默认值 | 说明                         |
|----------|----------|--------|------------------------------|
| `r18`   | `int`      | `0`      | `0`为非 R18，`1`为 R18，`2`为 r12 ，`3`为 全年龄，`4`为混合|            |
| `num`   | `int`   | `1`    | 返回的结果数量，范围为1到15，在库存不足或者指定搜索条件下结果数量可能会不足指定的数量 |
| `keyword`   | `string`  |    | 返回从标题、作者、标签中按指定关键字模糊匹配的结果   |
| `tag`   | `string[]`   |    | 返回匹配指定标签的作品   |
| `size`   | `string[]`   | `["original"]`   | 返回指定图片规格的地址   |
| `proxy`   | `string`   |  `i.pixiv.re`  | 设置图片地址所使用的在线反代服务   |
| `excludeAI`   | `boolean`   |  `false`  | 排除 AI 作品   |
| `dateAfter`   | `date`   |    | 返回在这个时间及以后上传的作品   |
| `dateBefore`   | `date`   |    | 返回在这个时间及以前上传的作品   |
