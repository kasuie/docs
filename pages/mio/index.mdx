# 前言

因为自己有个机器人，机器人涩图功能调用的第三方 api，大概是不满于涩图质量，以及想要拥有一个属于自己审美的涩图库，因此有了这个 api 的诞生 🎉

## 图片来源

初步规划的有两种来源，但绝大多数是来自[Pixiv](https://pixiv.net)，考虑到画师并没有发布在 P 站但又极想收录，所以少部分会通过存储在自建的图床，从而收录到本图库

## 图片收录

图片的收录，暂且只开放了 P 站的插画收录，每个人都可以在管理页面新增，为了保证图库的质量，设置了审核机制，但审核只针对插画收藏量小于 3000 的，
大于 3000 的可直接添加到图库，后续可能会调整，审核是我个人审核，所以难免会带有主观欣赏水平

## 额外说明

使用本 API 是你个人的行为，产生的任何问题都与我无关
