# remio-script-pixiv

## 推送排行榜接口

```js
POST `${BASE_URL}/prank/newDate`
```

| 参数名       | 类型  | 默认值 | 说明                                                                                     |
| :----------- | :-------- | :----- | :--------------------------------------------------------------------------------------- |
| `nextDate`   | `boolean`     |        | 明天的日期                             |
| `prevDate`  | `string`     |        | 昨天的日期 |   
| `rankDate`   | `string` |        |            今天的日期                            |   
| `rankList`    | [Rank](/script/remio-script-pixiv#rank-数据类型格式)[]     |        | 排行榜的作品数据                                                                  |
| `rankType`       | `string`  |        | 排行榜类型，示例：`daily` 表示每日排行，其他类型如`weekly`、`monthly`等                                                                 |
| `uploadName`       | `string`  |        | 上传者pixiv的用户名                                                         |

### `Rank` 参数详细说明

| 参数名          | 类型  | 默认值 | 说明                                                                                     |
| :-------------- | :-------- | :----- | :--------------------------------------------------------------------------------------- |
| `aspectRatio`   | `float`   |        | 图片的长宽比，表示宽度与高度的比值                  |
| `author`        | `string`  |        | 作者的名字                                                                |
| `createDate`    | `date`    |        | 创建日期，为`Date`对象               |
| `datePath`      | `string`  |        | 文件存储路径，年月日小时分钟秒的格式，例如：`2025/02/19/16/24/08`                    |
| `endDate`       | `string`  |        | 最多在榜两天，结束日期，格式为 `20250220_5:1`，代表今天排行为1，昨天排行为5                            |
| `exts`          | `string`  |        | 文件扩展名，表示图片的格式                                                   |
| `height`        | `int`     |        | 图片的高度，单位为像素                                                      |
| `illusType`     | `int`     |        | 图像类型                                                |
| `pageCount`     | `int`     |        | 作品一共有几张                                           |
| `pid`           | `int`     |        | 图片的唯一pid，示例：`127409639`                                                       |
| `pixAvatar`     | `string`  |        | 图片的头像，通常为一张缩略图，示例：`2024/10/10/20/19/26/xxx.jpg` |
| `rankType`      | `string`  |        | 排行类型，示例：`daily` 表示每日排行，其他类型如`weekly`、`monthly`等                   |
| `ratingCount`   | `int`     |        | 评分次数                                             |
| `startDate`     | `date`    | `null` | 最多在榜两天，开始日期，格式为 `20250220_0:2`，代表今天排行为2，0代表首次上榜                                                           |
| `status`        | `int`     |        | 作品收藏数                                       |
| `tags`          | `string`  |        | 图像的标签，多个标签用逗号分隔 |
| `title`         | `string`  |        | 图像的标题                                                            |
| `uid`           | `int`     |        | 作者用户的唯一标识符                                                         |
| `uploadName`    | `string`  |        | 上传者的名字                                                               |
| `uploadUid`     | `int`     | `null` | 上传者的唯一标识符                                                    |
| `viewCount`     | `int`     |        | 浏览次数                                                               |
| `width`         | `int`     |        | 图片的宽度，单位为像素                                                     |

## 推送作品接口

```js
POST `${BASE_URL}/img/save`
```

| 参数名          | 类型  | 默认值 | 说明                                                                                     |
| :-------------- | :-------- | :----- | :--------------------------------------------------------------------------------------- |
| `aiType`        | `int`     |        | 是否是 AI 作品，0 未知（旧画作或字段未更新），1 不是，2 是                                               |
| `aspectRatio`   | `float`   |        | 图片的长宽比，表示宽度与高度的比值                  |
| `author`        | `string`  |        | 作者的名字                                                      |
| `createDate`    | `date`    |        | 创建日期                                    |
| `datePath`      | `string`  |        | 文件存储路径，年月日小时分钟秒的格式，例如：`2025/02/20/07/30/01`                    |
| `ext`           | `array`   |        | 文件扩展名数组，表示图片的格式                                             |
| `height`        | `int`     |        | 图片的高度，单位为像素                                                  |
| `illustType`    | `int`     |        | 作品类型，`0`表示是插画，`1`表示是漫画                                     |
| `pageCount`     | `int`     |        | 作品一共有几张                                            |
| `pid`           | `int`     |        | 图片的唯一标识符pid                                                |
| `pixAccount`    | `string`  |        | 作者的pixiv账号                                                            |
| `pixAvatar`     | `string`  |        | 作者的pixiv头像，示例：`2013/06/12/00/22/23/xxx.jpg` |
| `r18`           | `int`     |        | 限制等级：0：r12，1：r18，2：全年龄，10：未分类                                                |
| `status`        | `int`     |        | 作品收藏数                                 |
| `tag`           | `string[]`|        | 标签数组                                                         |
| `tags`          | `string`  |        | 图像的标签，多个标签用逗号分隔                                          |
| `title`         | `string`  |        | 图像的标题                                                         |
| `uid`           | `int`     |        | 作者的唯一标识符                                                        |
| `uploadUid`     | `int`     | `-1`   | 上传者的唯一标识符                                                    |
| `uploadUserName`| `string`  |        | 上传者的名字`                                                              |
| `wallpaper`     | `int`     |        | 壁纸类型：0：未知，1：横屏，2：竖屏，3：头像                                                         |
| `width`         | `int`     |        | 图片的宽度，单位为像素                                                      |
