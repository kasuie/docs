# 油猴插件合集 ✨

插件开源地址 [Github](https://github.com/kasuie/remio-script) 包含：

- `remio-script-pixiv`: 主要是抓取网站的排行数据以及推送作品到 mio 图库
- `remio-script-bbs`: 主要是抓取原神角色武器数据
- `remio-script-bangumi`: 主要是抓取网站维护的番剧游戏等数据信息

基本都是抓取数据，然后把数据推送到自己的 api 接口进行保存维护，所以如果你也想要使用，需要保证脚本抓取到的数据格式要和你接口的数据格式保持一致(当然脚本是开源的，你如果能够自己修改也是可以的)。

## 脚本数据接口及格式

其中`${BASE_URL}`暂时只能通过环境变量配置是你自己的服务端地址(~~意思现在只能自己配置环境变量然后通过项目打包一个自己的脚本使用~~)
