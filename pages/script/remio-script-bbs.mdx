# remio-script-bbs

## 推送角色接口

```js
POST `${BASE_URL}/ys/updateRoles`
```

| 参数名       | 参数类型  | 默认值 | 说明                                                                                     |
| :----------- | :-------- | :----- | :--------------------------------------------------------------------------------------- |
| `area`       | `int`     |        | 0：未知、1：蒙德、2：璃月、3：稻妻、4：须弥、5：枫丹、6：至冬                             |
| `contentId`  | `int`     |        | 米游社content_id |
| `disabled`   | `boolean` |        | 是否禁用                                       |
| `element`    | `int`     |        | 0：未知、1：冰、2：草、3：火、4：水、5：风、6：雷、7：岩                                                                  |
| `icon`       | `string`  |        | 角色头像米游社的地址                                                                   |
| `role`       | `string`  |        | 角色名称                                                         |
| `star`       | `int`     |        | 角色星级                                                                           |
| `summary`    | `string`  |        | 角色简单介绍                                                           |
| `weaponType` | `int`     |        | 0：未知、1：单手剑、2：双手剑、3：弓、4：长柄武器、5：法器                                                         |


## 推送武器接口

```js
POST `${BASE_URL}/ys/updateWeapons`
```

| 参数名       | 类型  | 默认值 | 说明                                                                                     |
| :----------- | :-------- | :----- | :--------------------------------------------------------------------------------------- |
| `attrs`      | `string`  |        | 武器属性 ，多个方式以英文`,`分隔                             |
| `contentId`  | `int`     |        | 米游社content_id                 |
| `disabled`   | `boolean` |        | 是否禁用                                       |
| `getWay`     | `string`  |        | 获取方式，多个方式以英文`,`分隔                                      |
| `icon`       | `string`  |        | 武器头像米游社的地址                                                                   |
| `star`       | `int`     |        | 武器星级                                                                           |
| `summary`    | `string`  |        | 武器简单介绍                                                           |
| `weaponName` | `string`  |        | 武器名称                                                         |
| `weaponType` | `int`     |        | 0：未知、1：单手剑、2：双手剑、3：弓、4：长柄武器、5：法器                                                         |
